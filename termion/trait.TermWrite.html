Extension to the Write trait.

This extension to the Write trait is capable of producing the correct ANSI escape sequences for given commands, effectively controlling the terminal.

Required Methods

fn csi(&mut self, b: &[u8]) -> Result<usize>

Print the CSI (control sequence introducer) followed by a byte string.

fn osc(&mut self, b: &[u8]) -> Result<usize>

Print OSC (operating system command) followed by a byte string.

fn dsc(&mut self, b: &[u8]) -> Result<usize>

Print DSC (device control string) followed by a byte string.
